# berpikir dalam erlang


Repo ini berisi semua catatan pribadi tentang bahasa pemrograman Erlang/OTP

## Untuk mengunduh semua plugin silahkan buka terminal
----------------------------------------------------

Build
-----

```terminal
$ rebar3 compile
```

Build documentation
-------------------

```terminal
$ rebar3 ex_doc -o docs
```

Running shell
--------------

```terminal
$rebar3 shell
```
